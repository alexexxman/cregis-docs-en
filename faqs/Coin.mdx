## How to get the currency code?

`Cregis` Obtain supported currency information in two ways：<br/>
1、Reference documentation[currency-identifiers](/api-reference/currency-identifiers), this document will be updated regularly, but real-time performance is not guaranteed. <br/>
1、Reference API [support coins](/api-reference/request-apis/currency-query), this interface will return all currency data supported by the project in real time
